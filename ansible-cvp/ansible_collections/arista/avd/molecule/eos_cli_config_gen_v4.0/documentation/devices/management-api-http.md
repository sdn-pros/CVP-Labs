### Management API VRF Access

| VRF Name | IPv4 ACL | IPv6 ACL |
| -------- | -------- | -------- |
| default | ACL-API | - |
| MGMT | ACL-API | - |

### Management API HTTP Configuration

```eos
!
management api http-commands
   protocol https
   protocol https ssl profile SSL_PROFILE
   no protocol http
   no shutdown
   !
   vrf default
      no shutdown
      ip access-group ACL-API
   !
   vrf MGMT
      no shutdown
      ip access-group ACL-API
```
